
Folder name is used to identify the type of OS distribution
The order on iPXE will be dictionary order.
You can use upper case and lower case to separate two regions.

Format:
  {OS Distribution}[Non letters]*

  Note that OS distribution can be upper/lower case.

Valid examples:
sles
SLES
SLES-GA
SLES-SP4

RedHat
redhat.A
REDHAT=G
